=======
Removing a CA
=======

- Remove from website sliders
- Remove from PDF

Website Sliders

1.  Open the Website > Company Logos > illustrator file
2.  Select and delete company logo
3.  Arrange the following logos to fill gap by rotating back all logos one spot
4.  Save for Web > Percent 200% > Save into Images folder
5.  Open Cyberduck and FTP to website
6.  Navigate to /downloads/ca/sliders/
7.  Select and drag local images to Cyberduck to replace current files
8.  Push and commit changes to Website repository

CA List PDF

1.  Open Cyberduck and FTP to website
2.  Go to /downloads/ca/
3.  Go to UnitAdmin > O_EGI_Corporate... > EGI_CA_Program
4.  Check to make sure files have been updated and do not have the company that is being removed
5.  Select and drag all local pdf files to Cyberduck to replace current files
